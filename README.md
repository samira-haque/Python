# 🐍 Python Programming

A comprehensive collection of Python programming concepts, examples, and practice exercises designed for beginners who want to build a strong foundation in Python.

## 📚 Course Contents

### 📁 01 Variables and Datatype
Learn the fundamentals of Python:
- Variables
- Data Types
- Type Casting
- User Input
- Arithmetic Operations
- Basic Expressions

### 📁 02 Strings & Conditional Statements
Understand text processing and decision-making:
- String Operations
- String Methods
- String Slicing
- Comparison Operators
- Logical Operators
- if, elif, else Statements
- Nested Conditions

### 📁 03 List & Tuple
Explore Python's sequence data structures:
- Lists
- List Methods
- List Comprehension
- Tuples
- Tuple Operations
- Indexing & Slicing
- Iteration Techniques

### 📁 04 Dictionary & Set
Work with advanced collections:
- Dictionaries
- Dictionary Methods
- Nested Dictionaries
- Sets
- Set Operations
- Membership Testing
- Practical Examples

### 📁 05 Loops
Master repetitive tasks:
- for Loops
- while Loops
- break Statement
- continue Statement
- pass Statement
- Nested Loops
- Pattern Problems

### 📁 06 Function
Write reusable code:
- Function Definition
- Parameters & Arguments
- Return Statements
- Default Parameters
- Keyword Arguments
- Lambda Functions
- Variable Scope

### 📁 07 Recursion
Learn recursive problem-solving:
- Recursive Functions
- Base Cases
- Recursive Calls
- Factorial Program
- Fibonacci Series
- Recursive Patterns

### 📁 08 File Input Output
Handle files efficiently:
- Reading Files
- Writing Files
- Appending Data
- File Modes
- Exception Handling
- Context Managers (`with` statement)

## 🚀 Getting Started

### Prerequisites

- Python 3.x

Verify installation:

```bash
python --version
```

### Clone the Repository

```bash
git clone https://github.com/samira-haque/Python.git
cd Python
```

### Run Python Files

```bash
python filename.py
```
